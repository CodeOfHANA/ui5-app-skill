## [1.0.0] - 2026-02-16

### Added
- Initial release of UI5 App Skill documentation
- Complete SKILL.md with comprehensive SAPUI5/Fiori development standards
- README.md with quick start guide and overview
- MIT License
- Project structure and naming conventions
- MVC architecture and BaseController patterns
- Formatters and data binding guidelines
- Routing and navigation documentation
- i18n and localization standards
- OData v2 and v4 (RAP) integration guides
- UI5 CLI tooling reference
- Hungarian notation reference table
- Performance best practices
- Quick reference cheat sheets

### Features
- Project naming and application ID conventions
- Standard folder structure for Fiori freestyle apps
- Fiori Launchpad integration guidelines (semantic objects, actions, aliases)
- manifest.json configuration templates
- Component.js setup and initialization patterns
- BaseController with utility methods (getRouter, getModel, setModel, getResourceBundle, onNavBack)
- Formatter module patterns and XML view integration
- Routing configuration with parameter handling
- i18n properties file format with annotation prefixes
- Model conventions (default, i18n, view, device)
- OData v2 read, bind, filter, and write patterns
- OData v4/RAP context-based API
- RAP draft handling (Edit, Activate, Discard actions)
- RAP actions and function imports
- Deep entity navigation with $expand
- Fragment usage patterns
- JavaScript variable and function naming conventions
- XML view formatting conventions
- UI5 CLI commands for development and production builds
